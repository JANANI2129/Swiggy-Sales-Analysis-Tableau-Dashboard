# 🍽️ Swiggy Sales Analysis Dashboard

## 📌 Project Overview
This project is an interactive Tableau dashboard developed to analyze Swiggy sales performance. The dashboard provides insights into sales, profit, cost, order trends, restaurant performance, cuisine analysis, and city-wise sales.

## 🎯 Objectives
- Analyze overall sales performance.
- Monitor monthly order trends.
- Compare cost and profit.
- Identify top-performing restaurants.
- Analyze cuisine-wise costs.
- Evaluate city-wise sales.

## 🛠️ Tools Used
- Tableau
- Microsoft Excel

## 📊 Dashboard Features
- Total Profit
- Total Cost
- Total Orders
- Average Discount
- Monthly Order Trend
- Monthly Cost & Profit Analysis
- City-wise Gross Sales Analysis
- Restaurant-wise Sales Distribution
- Cuisine-wise Cost Analysis
- Profit Contribution by Restaurant
- Interactive Filters
